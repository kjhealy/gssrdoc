# Differences in class are an incentive

usclass6

## Source

General Social Survey https://gss.norc.org

## Details

Question Here are different opinions about social differences in this
country. Please tell me for each one whether you strongly agree,
somewhat, agree, somewhat disagree, or strongly disagree. READ EACH
STATEMENT. CIRCLE ONE CODE FOR EACH. F. Only if differences in income
and social standing are large enough is there an incentive for
individual effort.

## Values

- `[1]` strongly agree

- `[2]` somewhat agree

- `[3]` somewhat disagree

- `[4]` strongly disagree

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
