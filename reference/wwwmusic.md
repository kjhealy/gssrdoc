# Use the web to learn about music

wwwmusic

## Source

General Social Survey https://gss.norc.org

## Details

Question A. Do you ever use the Web to learn about, listen to, or
discuss topics related to any kind of music?

## Values

- `[1]` yes

- `[2]` no

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
