# How able is x to decide receive treatment

dectreat

## Source

General Social Survey https://gss.norc.org

## Details

Question In your opinion, how able is NAME to make his/her own decisions
about the treatment s/he should receive – very able, somewhat able, not
very able or not able at all?

## Values

- `[1]` very able

- `[2]` somewhat able

- `[3]` not very able

- `[4]` not able at all

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
