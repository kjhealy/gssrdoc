# Would guess or type site name for information o

caninf4

## Source

General Social Survey https://gss.norc.org

## Details

Question Imagine you wanted to use the Web to find out information about
a candidate for political office. What would you do? D. Would guess or
type site name for information on candidate.

## Values

- `[0]` no

- `[1]` yes

- `[2]` yes, at least one url is not a .com

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
