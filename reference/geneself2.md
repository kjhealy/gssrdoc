# R would want genetic test done on fetus

geneself2

## Source

General Social Survey https://gss.norc.org

## Details

Question Today, tests are being developed that make it possible to
detect serious genetic defects before a baby is born. But so far, it is
impossible either to treat or to correct most of them. If (you were
pregnant/you were expecting a child/your partner were pregnant), would
you want (the pregnant woman/her) to have a test to find out if the baby
has any serious genetic defects?
