# Happy if daughter engineer

engda

## Source

General Social Survey https://gss.norc.org

## Details

Question A. If you had a daughter, how would you feel if she wanted to
be an engineer—would you feel happy, unhappy, or would you not care one
way or the other?

## Values

- `[1]` happy

- `[2]` not care

- `[3]` unhappy

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job

  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
