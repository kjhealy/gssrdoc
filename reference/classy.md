# Subjective social class-version y

classy

## Source

General Social Survey https://gss.norc.org

## Details

Question B. If you were asked to use one of five names for your social
class, which would you say you belong in: the lower class, the working
class, the middle class, the upper middle class, or the upper class?

## Values

- `[1]` lower class

- `[2]` working class

- `[3]` middle class

- `[4]` upper middle class

- `[5]` upper class
