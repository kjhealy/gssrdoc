# R preference re: firm size in

smallbig

## Source

General Social Survey https://gss.norc.org

## Details

Question Suppose you were working and could choose between different
kinds of jobs. Which of the following would you personally choose? I
would advise... Please CIRCLE ONE CODE FOR EACH. B. Working in a small
firm (or) working in a large firm

## Values

- `[1]` small

- `[2]` big

- `[NA(d)]` can't choose `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
