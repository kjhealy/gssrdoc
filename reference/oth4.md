# Need but unable to use public transport.

oth4

## Source

General Social Survey https://gss.norc.org

## Details

Question Now, I'm going to ask you about various events and conditions
that happen to people. I'm interested in those that happened to you
during the last 12 months, that is since (CURRENT MONTH), (1990/2003).
As I ask you about the specific events, please think carefully, so I can
record things accurately. H. And finally, since (CURRENT MONTH),
(1990/2003), did any of the following occur to you? 4. Needed, but
unable to use, public transportation to get to work, shop, etc.

- `[2]` no
