# Size of place in 1000s

size

## Source

General Social Survey https://gss.norc.org

## Details

Question D. SIZE OF PLACE Size of Place in thousands-A 4-digit number
which provides actual size of place of interview. Remember when using
this code to add 3 zeros. Listed below are the frequencies for gross
population categories.

## Values

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
