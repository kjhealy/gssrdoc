# Immigration from europe should be increased

letineur

## Source

General Social Survey https://gss.norc.org

## Details

Question What about the number of immigrants from Europe? (should it be
increased a lot, increased a little, left the same as it is now,
decreased a little, or decreased a lot?)

## Values

- `[1]` increased a lot

- `[2]` increased a little

- `[3]` left the same as it is now

- `[4]` decreased a little

- `[5]` decreased a lot

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
