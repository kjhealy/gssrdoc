# Use web to learn about literature

wwwlit

## Source

General Social Survey https://gss.norc.org

## Details

Question A. Do you ever use the Web to learn about, read, or discuss
topics related to literature"novels, poetry, or plays?

## Values

- `[1]` yes

- `[2]` no

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
