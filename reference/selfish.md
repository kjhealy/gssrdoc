# R is a selfish person

selfish

## Source

General Social Survey https://gss.norc.org

## Details

Question For each of the following, please indicate how well the
description applies to you by circling one number: E. A selfish person.
Is this ……

## Values

- `[1]` a very good description

- `[2]` a good description

- `[3]` a fair description

- `[4]` not a very good description

- `[5]` not a good description at all

- `[9]` refused

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
