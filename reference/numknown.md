# Variable that drives skips in module

numknown

## Source

General Social Survey https://gss.norc.org

## Details

Question Cases in Sample B (Ballots 4-6) should receive this variable.
This variable is used to drive skips in a module. This variable should
have a value of 1-5 using the following proportions: 1=50pct, 2,3,4,5 =
12.5pct each

## Values

- `[1]` subset1

- `[2]` subset2

- `[3]` subset3

- `[4]` subset4

- `[5]` subset5

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
