# How many years and months since 2nd closest suicide

suitime2

## Source

General Social Survey https://gss.norc.org

## Details

Question How many months and years has it been since that person's
death? IF RESPONDENT PROVIDES AT LEAST A YEAR AMOUNT, PLEASE SELECT THE
FIRST OPTION.

## Values

- `[1]` respondent provides number of months or years

- `[2]` respondent provides verbatim response uncodeable in months/years

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
