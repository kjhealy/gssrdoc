# Which group is r most active in

mostact

## Source

General Social Survey https://gss.norc.org

## Details

Question Which one of these groups or organizations (READ GROUP FROM
MEMFRAT THAT R REPLIED "YES" TO) are you most active in? CODE ONLY ONE.
