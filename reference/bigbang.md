# Sci knowledge: the universe began with a huge explosion

bigbang

## Source

General Social Survey https://gss.norc.org

## Details

Question Now, I would like to ask you a few short questions like those
you might see on a television game show. for each statement that I read,
please tell me if it is true or false. If you don't know or aren't sure,
just tell me so, and we will skip to the next question. Remember true,
false, or don't know. G. The universe began with a huge explosion. (Is
that true or false?)

## Values

- `[1]` true

- `[2]` false

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
