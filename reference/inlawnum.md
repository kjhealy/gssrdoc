# Number of in-laws

inlawnum

## Source

General Social Survey https://gss.norc.org

## Details

Question Now thinking of all your adult relatives – those still living
and age 18 or older. How many of each do you have? READ EACH ITEM. CODE
A NUMBER OR ZERO FOR EACH. D. Parents-in-Law and Adult Brothers- and
Sisters-in-Law

## Values

- `[96]` 96 or more

- `[97]` \# dk but ) 0

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
