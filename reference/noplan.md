# There's no sense planning a lot

noplan

## Source

General Social Survey https://gss.norc.org

## Details

Question The following are some statements that people have made. For
each one, please tell me if you strongly agree, agree, disagree, or
strongly disagree with the statement. (NOTE: INTERVIEWERS SHOULD CODE
"neither agree or disagree/it depends" ONLY WHEN VOLUNTEERED.) A.
There's no sense planning a lot – if something good is going to happen,
it will.

## Values

- `[1]` strongly agree

- `[4]` disagree

- `[5]` strongly disagree
