# How serious x s situation is

mhproblm

## Source

General Social Survey https://gss.norc.org

## Details

Question Please remember, there are no right-or-wrong answers to these
questions. Please think about the person I just described when answering
these questions. First, how serious would you consider (NAME)'s
situation to be. would you say…

## Values

- `[1]` very serious

- `[2]` moderately serious

- `[3]` not very serious

- `[4]` not at all seriou

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
