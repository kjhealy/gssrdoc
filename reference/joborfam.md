# Would you remain in unsatisfying job for family

joborfam

## Source

General Social Survey https://gss.norc.org

## Details

Question (HAND CARD D3) Have you ever remained or would you remain in a
job that was not satisfying for you for the benefit of your family life?

## Values

- `[1]` yes, i have done so and probably would do so again

- `[2]` yes, i have done so but probably would not do so again

- `[3]` no, i have not done so but probably would do so

- `[4]` no, i have not done so and probably would not do so

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
