# Astrology has some scientific truth

scitest3

## Source

General Social Survey https://gss.norc.org

## Details

Question For each statement below, just check the box that comes closest
to your opinion of how true it is. In your opinion, how true is this? C.
Astrology - the study of star signs - has some scientific truth.

## Values

- `[1]` definitely true

- `[2]` probably true

- `[3]` probably not true

- `[4]` definitely not true

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
