# Allow workers more flexible hours

flextime

## Source

General Social Survey https://gss.norc.org

## Details

Question It is becoming more common for employers to offer ways for
people to combine work and being a parent. For each of the following,
please tell me whether or not you think that employers should offer
these options to their workers. A. Allow workers more flexible working
hours, to handle family responsibilities.

## Values

- `[1]` yes, should offer this option

- `[2]` no, should not offer this option

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
