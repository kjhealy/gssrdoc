# Thought one was someplace before

dejavu

## Source

General Social Survey https://gss.norc.org

## Details

Question How often have you had any of the following experiences? READ
EACH ITEM AND CIRCLE ONE CODE FOR EACH. A. Thought you were somewhere
you had been before, but knew that it was impossible.

## Values

- `[1]` never in my life

- `[2]` once or twice

- `[3]` several times

- `[4]` often

- `[8]` cannot answer

- `[NA(d)]` can't choose `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
