# R would want to abort fetus if there were genetic defects

geneabrt2

## Source

General Social Survey https://gss.norc.org

## Details

Question Suppose a test shows the baby has a serious genetic defect.
Would you (yourself want/want the pregnant woman/want your partner) to
have an abortion if a test shows the baby has a serious genetic defect?
