# Visit sites about music not usually heard

musicnew

## Source

General Social Survey https://gss.norc.org

## Details

Question Is it VERY TRUE for you, SOMEWHAT TRUE, or NOT TRUE that: C.
You visit sites where you can learn about or hear types of music that
you wouldn't ordinarily listen to?

## Values

- `[1]` very true

- `[2]` somewhat true

- `[3]` not true

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
