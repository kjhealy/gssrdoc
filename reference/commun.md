# Feelings about communism

commun

## Source

General Social Survey https://gss.norc.org

## Details

Question Thinking about all the different kinds of governments in the
world today, which of these statements comes closest to how you feel
about Communism as a form of government? CODE ONE.

## Values

- `[1]` worst kind

- `[2]` bad, not worst

- `[3]` ok for some

- `[4]` good form

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
