# I could get out of a jam

hope1

## Source

General Social Survey https://gss.norc.org

## Details

Question (HAND CARD G2) Using the scale on the showcard, please select
the response that best describes how you think about yourself right now.
Please focus on yourself and what is going on in your life at this
moment. Please answer each item as I read them according to the scale on
the card. If I should find myself in a jam, I could think of many ways
to get out of it.

## Values

- `[1]` definitely false

- `[2]` mostly false

- `[3]` somewhat false

- `[4]` slightly false

- `[5]` slightly true

- `[6]` somewhat true

- `[7]` mostly true

- `[8]` definitely true

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
