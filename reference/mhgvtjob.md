# Govt responsibility: provide job for x

mhgvtjob

## Source

General Social Survey https://gss.norc.org

## Details

Question The next few questions deal with the government's
responsibility to help people like (NAME). For each statement please
tell me if you think the government definitely should, probably should,
probably should not, or definitely should not be responsible for helping
people with situations like (NAME). A. First, provide a job for (NAME)
if (NAME) wants one.

## Values

- `[1]` definitely should be

- `[2]` probably should be

- `[3]` probably should not be

- `[4]` definitely should not be

- `[NA(d)]` can't choose `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
