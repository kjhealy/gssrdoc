# Hospitals should mainly be run by private

runhosp

## Source

General Social Survey https://gss.norc.org

## Details

Question Do you think each of the following should mainly be run by
private organizations or companies, or by the government? PLEASE CHECK
ONE BOX ON EACH LINE. B. Hospitals
