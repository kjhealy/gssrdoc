# R's current immigration status right now

immstats

## Source

General Social Survey https://gss.norc.org

## Details

Question Thinking of your current immigration status right now, is that
in Box A, B, or C? If it's in Box B, we do not want to know which
specific category applies to you, and there will be no more questions
about Box B. (IF RESPONDENT ASKS WHY, SAY: Because right now we are
focusing on Box A.)

## Values

- `[1]` box a

- `[2]` box b

- `[3]` box c

- `[4]` other

- `[NA(d)]` don't know `[NA(i)]` iap `[NA(j)]` I don't have a job
  `[NA(m)]` dk, na, iap `[NA(n)]` no answer `[NA(p)]` not imputable
  `[NA(r)]` refused `[NA(s)]` skipped on web `[NA(u)]` uncodeable
  `[NA(x)]` not available in this release `[NA(y)]` not available in
  this year `[NA(z)]` see codebook
